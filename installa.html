Installazione in 2 minuti

Aggiungere il venditore AI al tuo sito è una sola riga di codice. Cambia solo dove incollarla a seconda della piattaforma. Qui sotto trovi il punto esatto per ciascuna.

① La riga da incollare

È quella che lo Studio ti genera per ogni cliente (bottone «💾 Salva cliente & codice d'imbarco»). Ha questa forma:

<script src="https://aiseller.pages.dev/widget.js" defer></script>
📍 Regola d'oro: questa riga è uguale per tutti i clienti — non cambia mai. Incollala una volta sola nel footer / template globale del sito, subito prima di </body> → appare su tutte le pagine automaticamente. Il widget riconosce il dominio da solo.

🌐 Sito in HTML manuale

  1. Apri il file del template comune (di solito footer.html o il fondo di ogni pagina).
  2. Incolla la riga subito prima di </body>.
  3. Salva e ricarica online. Fatto.

🟦 WordPress il più diffuso

Modo consigliato (senza toccare codice):

  1. Installa il plugin gratuito WPCode (oppure «Insert Headers and Footers»).
  2. Vai su WPCode → Add Snippet → Custom HTML.
  3. Incolla la riga, imposta posizione «Footer» / «Before </body>», modalità Auto-insert.
  4. Save & Activate. Esce su tutto il sito.

In alternativa (tema): Aspetto → Editor file del tema → footer.php, incolla prima di </body> (meglio su un child theme).

🛒 Shopify

  1. Online Store → Themes → sul tema attivo: ⋯ → Edit code.
  2. Apri theme.liquid.
  3. Incolla la riga subito prima di </body> → Save.

Appare su tutto lo store (home, prodotti, collezioni, checkout escluso).

🟣 Wix · 🟧 Squarespace · 🔵 Webflow

Wix
Settings → Custom Code → + Add → incolla → posizione «Body - end» → All pages → Apply.
Squarespace
Settings → Advanced → Code Injection → campo «Footer» → incolla → Save.
Webflow
Project Settings → Custom Code → «Footer Code» → incolla → Save → Publish.
Altri (Joomla, Prestashop…)
Cerca «footer» / «codice personalizzato» / «before body» nel pannello e incolla lì.

🏷️ Google Tag Manager universale

Se il sito usa GTM, funziona ovunque senza toccare il tema:

  1. In GTM: Tag → New → Tag Configuration → Custom HTML.
  2. Incolla la riga.
  3. Triggering → All Pages.
  4. Save → Submit → Publish.

📍 Posizione & convivenza con altri bottoni

Se il sito ha già WhatsApp o Chiamata flottanti, la pallina li rileva e si mette sopra da sola. Puoi anche regolare a mano:

  • data-position="left" → la sposta sull'altro lato (separazione netta)
  • data-bottom="100" → la alza di X px
  • data-avoid="off" → disattiva lo spostamento automatico

✅ Verifica finale

  1. Apri il sito (in incognito). La pallina col personaggio compare in basso a destra.
  2. Non la vedi? Fai un hard refresh (Ctrl/Cmd+Shift+R): è quasi sempre la cache.
  3. Cliccala: si apre la chat. 🎉
